# 📘 Assignment: Data Structures in Python

## 🎯 Objective

Practice using Python lists, dictionaries, sets, and tuples to store and process data. By the end, you will be able to choose the right data structure for a specific problem.

## 📝 Tasks

### 🛠️ Build Core Collection Examples

#### Description
Create short Python examples that demonstrate how each data structure works. Use one real-world theme (for example: student clubs, game scores, or a school library) across all examples.

#### Requirements
Completed program should:

- Create and update a list with at least 5 items
- Create a dictionary with at least 3 key-value pairs and print one value by key
- Create a set and show how duplicate values are removed
- Create a tuple and access values by index
- Print clear labels so a reader can identify each data structure in the output


### 🛠️ Solve a Data Challenge

#### Description
Use all four data structures to solve a small challenge: organize student activity sign-ups, remove duplicate names, and generate a final summary.

#### Requirements
Completed program should:

- Start with a list of student names that includes duplicates
- Use a set to produce a unique participant list
- Use a dictionary to map each participant to a chosen activity
- Store fixed metadata in a tuple (for example: assignment name and class period)
- Print a final summary showing total sign-ups, unique participants, and activity assignments
