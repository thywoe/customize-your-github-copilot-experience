# 📘 Assignment: Games in Python

## 🎯 Objective

Build a simple Hangman game in Python. In this assignment, you will practice using variables, loops, conditionals, strings, and user input to create an interactive program.

## 📝 Tasks

### 🛠️	Set Up the Game

#### Description
Start by preparing the main pieces of the Hangman game. Choose a secret word from the provided word list and create the variables needed to track the player's guesses and remaining attempts.

#### Requirements
Completed program should:

- Randomly select one word from the predefined list.
- Create variables to store guessed letters and the number of incorrect guesses.
- Set a maximum number of incorrect guesses allowed.


### 🛠️	Build the Gameplay Loop

#### Description
Write the main game loop so the player can keep guessing letters until they either reveal the whole word or run out of attempts. Show the player's progress after each turn and print a final win or lose message.

#### Requirements
Completed program should:

- Display the hidden word using underscores for letters that have not been guessed yet.
- Accept a letter guess from the user and update the game state based on whether the guess is correct.
- End the game with a clear win message if the word is guessed or a lose message if the player uses all attempts.
